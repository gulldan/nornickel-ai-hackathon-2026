# Документация

Этот раздел хранит подробности, которые не нужны в корневом README.

Корневой README отвечает на вопрос, что это за продукт и как его запустить.
Документы ниже раскрывают устройство, сценарии и ограничения.

## Быстрый выбор

1. Нужно запустить стенд: [DEPLOY-QUICKSTART.md](DEPLOY-QUICKSTART.md).
2. Нужно понять продуктовые экраны:
   [product/PRODUCT_PAGES_AND_IDEAS.md](product/PRODUCT_PAGES_AND_IDEAS.md).
3. Нужно объяснить генерацию гипотез:
   [product/HYPOTHESIS_GENERATION.md](product/HYPOTHESIS_GENERATION.md).
4. Нужно объяснить поиск и ответы с источниками:
   [product/RAG_BUSINESS_TECHNICAL_OVERVIEW.md](product/RAG_BUSINESS_TECHNICAL_OVERVIEW.md).
5. Нужно объяснить алгоритмы без лишней инфраструктуры:
   [product/ALGORITHMS_AND_ML_MODELS.md](product/ALGORITHMS_AND_ML_MODELS.md).
6. Нужно разобраться в сервисах:
   [ARCHITECTURE.md](ARCHITECTURE.md).
7. Нужно провести демо:
   [DEMO-CASE-TOF.md](DEMO-CASE-TOF.md).
8. Нужно проверить JSON-схемы:
   [schemas/README.md](schemas/README.md).

Папка `docs/product/presentation` содержит материалы презентации.
